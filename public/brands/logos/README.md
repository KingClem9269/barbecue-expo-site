# Logos des marques exposantes

Déposer ici les logos au format **SVG** (ou PNG avec fond transparent).

Convention de nommage : `<slug>.svg` où `slug` est celui défini dans
`content/brands/index.json`.

Exemple :
- `weber.svg`
- `traeger.svg`
- `kamado-joe.svg`
- `ofyr.svg`
- …

## Fallback

Si le logo n'est pas trouvé, la carte affiche le nom de la marque en
typographie SansPlomb (fallback automatique, aucune action requise).

## Taille recommandée

- Ratio **5:3** environ (zone d'affichage dans la carte)
- **Fond transparent** obligatoire (pas de carré blanc)
- Résolution : SVG vectoriel préféré, sinon PNG 720x432px minimum

## Où récupérer les logos

- Site officiel de chaque marque (press kit usuel)
- Barbecue-expo.fr (iframe Imagina → clic sur la marque → logo affiché)
